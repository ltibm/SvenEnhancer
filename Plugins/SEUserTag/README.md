# Description
This plugin allows assigning tags to player names and coloring them in the chat section. It also applies coloring in the score info section.
You can use the **se_reloadtags** command from both the client and server console to reload the list.


# Adding Users
Open **se_usertags.toml** from scripts/plugins/config
Examples are included inside.

# Requirements
SvenEnhancer Min v1.01

# Installation

append to default_plugins.txt
```
	"plugin"
	{
		"name" "SE User Tag"
		"script" "se_usertags"
	}
```
